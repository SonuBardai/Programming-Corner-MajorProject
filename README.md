# Programming Corner - A Real-time Chatting and Coding Application

-   Programming Corner is a hub for programmers to find solutions to their programming needs.
-   It is a full-stack web application that provides a community for programmers to share and learn different subjects of Information Technology.
-   Our web-based interactive platform aids coders and tech-enthusiasts to teach others and create an environment for themselves to improve their technical abilities.
-   It is a one-stop center for programming aficionados to access tech solutions and overcome all the hurdles that they come across in their coding journeys.
-   Programming Corner provides a wide range of modules that facilitate coders to boost their programming skills.

## Technologies Used

-   HTML, CSS, TailwindCSS, JS
-   NODE JS, EXPRESS JS, SOCKET IO
-   MongoDB

## Modules

-   Resume Builder
-   Group Chat
-   Blogging
-   Online Code Compiler
-   Authentication System
